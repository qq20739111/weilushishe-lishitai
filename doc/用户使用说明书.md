## 1. 前言

### 1.1 系统简介

「围炉诗社·理事台」是一套专为诗社日常管理设计的智能管理系统。系统运行在一个小型智能硬件设备上，通过浏览器即可访问使用，无需安装任何软件。

主要功能包括：
- **诗歌管理**：发布、浏览、管理诗词作品
- **活动管理**：发起和管理诗社活动
- **事务管理**：任务分配、认领与积分奖励
- **财务管理**：收支记录与统计公示
- **成员管理**：社员信息维护与权限管理
- **实时聊天**：「摆龙门阵」在线交流功能

### 1.2 适用对象

本说明书适用于围炉诗社全体成员，包括普通社员、理事、财务人员及管理员。不同角色可使用的功能有所不同，详见[权限说明](#8-权限说明)章节。

### 1.3 设备说明

系统运行在一个白/黑色小型设备上，设备上有一个蓝色指示灯用于显示工作状态。设备需要连接电源（USB供电）才能工作。

## 2. 系统启动与指示灯

### 2.1 设备启动流程

1. **接通电源**：将设备通过USB线连接电源
2. **等待启动**：设备开始自动启动，蓝色指示灯会开始快速闪烁
3. **连接网络**：设备尝试连接预设的WiFi网络
4. **启动完成**：指示灯变为缓慢呼吸或熄灭，表示启动成功

整个启动过程通常需要 **30秒至2分钟**，请耐心等待。

### 2.2 指示灯状态说明

设备通过蓝色LED指示灯的不同闪烁方式来表示当前工作状态：

| 指示灯状态 | 闪烁速度 | 含义说明 |
|-----------|---------|---------|
| **快速呼吸** | 每秒闪烁2次 | 正在连接WiFi网络，请等待 |
| **中速呼吸** | 每秒闪烁1次 | 热点模式已启动，可通过热点连接 |
| **缓慢呼吸** | 每2秒闪烁1次 | WiFi连接成功，系统正常运行 |
| **极慢呼吸** | 每4秒闪烁1次 | 双模式运行（WiFi和热点同时工作） |
| **快闪一下** | 闪一下即停 | 系统正在响应您的操作 |
| **指示灯熄灭** | 不闪烁 | 系统正常运行中（节能模式） |

> **提示**：系统启动成功后，指示灯会在约60秒后自动熄灭以节省能源，这是正常现象。当您在网页上进行操作时，指示灯会快闪一下作为响应反馈。

### 2.3 异常状态识别

| 异常现象 | 可能原因 | 处理方法 |
|---------|---------|---------|
| 指示灯完全不亮 | 未通电或设备故障 | 检查电源连接 |
| 持续快速闪烁超过3分钟 | WiFi连接失败 | 检查WiFi名称和密码配置 |
| 设备反复重启 | 系统异常 | 断电等待30秒后重新上电 |

## 3. 连接与登录

### 3.1 访问方式一：通过家庭WiFi（推荐）

当设备成功连接到您的家庭WiFi后：

1. 确保您的手机或电脑连接到**同一个WiFi网络**
2. 打开浏览器（推荐使用Chrome、Safari或微信内置浏览器）
3. 在地址栏输入设备的IP地址，例如：`http://192.168.30.68`
4. 即可看到系统登录页面

> **如何获取设备IP地址？** 请联系管理员获取，或在路由器管理页面中查看已连接设备列表。

### 3.2 访问方式二：通过热点模式

当设备无法连接WiFi或首次配置时，会自动开启热点模式：

1. 在手机WiFi设置中，查找名为「**围炉诗社理事台**」的热点
2. 输入热点密码连接（默认密码请联系管理员）
3. 连接成功后，打开浏览器访问：`http://192.168.1.68`
4. 即可进入系统

> **注意**：连接热点后，手机将暂时无法访问互联网，这是正常现象。使用完毕后请切换回正常WiFi网络。

### 3.3 用户登录

1. 在登录页面输入您的**手机号**（11位）
2. 输入您的**登录密码**
3. 点击「登录」按钮
4. 登录成功后自动进入系统首页

> **首次登录**：新社员的初始密码由管理员设置并告知，首次登录后建议立即修改密码。

### 3.4 游客访问

如果系统开启了游客访问功能，未登录用户可以：
- 浏览首页内容
- 查看诗歌作品
- 查看活动信息
- 参与「摆龙门阵」聊天（如已开启游客聊天）

但以下功能需要登录后才能使用：
- 发布诗歌
- 认领任务
- 查看财务明细
- 修改个人信息

### 3.5 忘记密码

如果忘记密码，请联系诗社管理员进行密码重置。管理员可以在「社员管理」中为您设置新密码。

## 4. 功能模块使用指南

### 4.1 首页

首页是系统的主界面，包含以下内容：

**年度热力图**
- 以日历形式展示全年诗词创作情况
- 颜色越深表示当周创作越活跃
- 可切换年份查看历史数据

**近期活动**
- 显示最近的诗社活动
- 点击可查看活动详情

**最新诗作**
- 展示最新发布的诗词作品
- 点击可阅读全文

**排行榜**
- 显示本年度积分排名
- 积分通过完成任务获得

**今日推荐**
- 每日随机推荐一首诗作

**摆龙门阵**
- 快捷聊天入口
- 可直接发送消息

---

### 4.2 藏诗阁（诗歌管理）

「藏诗阁」是诗词作品的集中展示区。

#### 浏览诗歌
- 默认按时间倒序显示所有诗作
- 可使用搜索框搜索标题或内容
- 可按诗歌类型筛选（古体诗、绝句、律诗、词、现代诗等）
- 点击诗歌卡片可查看完整内容

#### 发布新作（需登录）
1. 点击「发布新作」按钮
2. 填写诗歌标题（必填）
3. 输入诗歌正文（必填）
4. 选择诗歌类型
5. 设置发布时间（可选，默认为当前时间）
6. 点击「发布」完成

#### 编辑/删除（仅限作者）
- 在诗歌详情页，作者可看到「编辑」和「删除」按钮
- 点击「编辑」可修改诗歌内容
- 点击「删除」需二次确认后删除

> **草稿功能**：编辑中的诗歌会自动保存到本地草稿，关闭页面后下次打开可继续编辑。

---

### 4.3 活动

「活动」模块用于管理诗社的各类活动。

#### 查看活动
- 活动列表显示所有活动
- 可按状态筛选：筹备中、报名中、进行中、已结束
- 点击活动卡片查看详细信息

#### 活动状态说明
| 状态 | 含义 |
|-----|------|
| 筹备中 | 活动正在策划准备 |
| 报名中 | 活动开放报名 |
| 进行中 | 活动正在举行 |
| 已结束 | 活动已完成 |

#### 发起活动（理事及以上）
1. 点击「发起活动」按钮
2. 填写活动主题（必填）
3. 输入活动详情说明
4. 设置活动时间（必填）
5. 填写活动地点
6. 选择活动状态
7. 点击「发布」完成

---

### 4.4 摆龙门阵（聊天）

「摆龙门阵」是诗社成员的实时聊天室。

#### 基本使用
1. 点击导航栏「摆龙门阵」进入聊天室
2. 在底部输入框输入消息内容
3. 点击「发送」或按回车键发送
4. 消息会实时显示在聊天区域

#### 界面说明
- **右侧/主区域**：聊天消息列表
- **左侧边栏**：在线用户列表、聊天室状态
- 消息显示发送者昵称和发送时间

#### 注意事项
- 单条消息最多1024个字符
- 请文明发言，尊重他人
- 游客可参与聊天（如管理员已开启）

---

### 4.5 事务（任务管理）

「事务」模块用于诗社任务的发布、认领和管理。

#### 查看任务
- 任务列表显示所有事务
- 每个任务显示标题、状态、奖励积分
- 点击可查看任务详情

#### 任务状态流程
```
待认领 → 已认领 → 待审批 → 已完成
                      ↓
                    已拒绝
```

#### 认领任务（普通社员）
1. 找到状态为「待认领」的任务
2. 点击「认领」按钮
3. 任务状态变为「已认领」

#### 完成任务
1. 完成任务要求的工作
2. 点击「提交完成」按钮
3. 等待理事审批

#### 获得积分
- 任务审批通过后，奖励积分自动发放
- 可在个人中心查看积分余额

#### 发布任务（理事及以上）
1. 点击「发布事务」按钮
2. 填写任务标题和描述
3. 设置奖励积分
4. 可指定具体承接人（可选）
5. 点击「发布」完成

---

### 4.6 财务

「财务」模块公开展示诗社的收支情况。

#### 查看财务
- 顶部显示：当前余额、本年度收入、本年度支出
- 下方为详细的收支记录列表
- 每条记录显示：日期、摘要、金额、经办人

#### 记账（财务人员）
1. 点击「记账」按钮
2. 选择类型：收入或支出
3. 选择类别：会费、活动费用、物资采购等
4. 输入金额（必填）
5. 填写摘要说明（必填）
6. 选择记账日期
7. 填写经办人
8. 点击「保存」完成

---

### 4.7 社员

「社员」模块展示诗社成员名录。

#### 查看社员
- 以卡片形式展示所有社员
- 每张卡片显示：姓名、雅号、角色、积分
- 可查看社员的生日等信息

#### 管理社员（管理员）
- 点击「录入新社员」添加成员
- 点击社员卡片上的「编辑」修改信息
- 点击「删除」移除社员（需确认）

---

### 4.8 个人中心

点击导航栏右侧的个人头像或名字进入个人中心。

#### 查看个人信息
- 显示您的姓名、角色、积分等基本信息

#### 修改资料
1. 可修改「雅号」（花名/笔名）
2. 可设置「生日」
3. 点击「保存资料」生效

#### 修改密码
1. 输入原密码（当前使用的密码）
2. 输入新密码（至少6位，建议包含数字和字母）
3. 再次输入新密码确认
4. 点击「修改密码」完成

> **安全建议**：请定期修改密码，不要使用过于简单的密码如"123456"。

## 5. 系统管理（管理员专用）

以下功能仅管理员及以上角色可见和使用。

### 5.1 进入系统管理

登录后点击导航栏的「系统」进入管理后台。

### 5.2 设备状态面板

显示设备实时运行信息：
- **系统时间**：当前时间
- **运行时长**：设备已持续运行的时间
- **存储空间**：数据存储使用情况
- **运行内存**：内存使用情况
- **WiFi信号**：当前网络信号强度

### 5.3 数据统计

显示各模块的数据量：
- 社员数量
- 诗作数量
- 活动数量
- 事务数量
- 财务记录数

### 5.4 基础设置（理事及以上）

**系统名称**
- 可修改系统显示的名称
- 修改后点击「保存」生效

**积分名称**
- 可自定义积分的称谓，如「围炉值」
- 修改后全站生效

**功能开关**
| 设置项 | 说明 |
|-------|------|
| 开启网站访问 | 关闭后仅管理员可访问 |
| 允许游客访问 | 关闭后必须登录才能浏览 |
| 摆龙门阵功能 | 启用或禁用聊天室 |
| 龙门阵游客上限 | 允许同时聊天的游客数量(0-10) |
| 龙门阵人数上限 | 聊天室最大人数(5-100) |

### 5.5 WiFi配置（管理员及以上）

**连接家庭WiFi**
- WiFi名称：输入您的WiFi名称
- WiFi密码：输入WiFi密码
- IP获取方式：选择自动获取或手动设置静态IP

**热点设置**
- 热点名称：设备作为热点时的名称
- 热点密码：连接热点需要的密码

> **重要提示**：修改WiFi配置后需要重启设备才能生效。错误的WiFi配置可能导致无法连接设备，请谨慎操作。

### 5.6 自定义字段管理（理事及以上）

可为社员资料添加自定义信息字段，如：
- 入社日期
- 联系邮箱
- 个人简介等

**添加新字段**
1. 输入字段名称
2. 选择字段类型（文本、数字、日期、邮箱等）
3. 设置是否必填
4. 点击「添加」

### 5.7 登录日志（理事及以上）

查看最近的用户登录记录，包括：
- 登录用户名称
- 登录时间
- 登录状态（成功/失败）

## 6. 数据备份与恢复（超级管理员专用）

此功能仅超级管理员可使用，用于保护和恢复系统数据。

### 6.1 为什么要备份

- 防止设备故障导致数据丢失
- 更换设备时迁移数据
- 定期存档保留历史记录

### 6.2 导出备份

1. 进入「系统」→ 找到「数据备份」区域
2. 点击「下载备份文件」按钮
3. 系统开始打包所有数据
4. 等待进度条完成
5. 浏览器自动下载备份文件（JSON格式）

**备份包含的数据：**
- 所有社员信息
- 全部诗词作品
- 活动记录
- 事务任务
- 财务记录
- 积分日志
- 系统设置

> **建议**：每月至少备份一次，重要数据更新后及时备份。

### 6.3 导入恢复

1. 进入「系统」→ 找到「数据备份」区域
2. 点击「选择备份文件」
3. 选择之前下载的备份文件（.json格式）
4. 系统提示确认（此操作会覆盖现有数据）
5. 确认后等待导入完成

> **警告**：导入备份会覆盖当前所有数据，请确保已备份当前数据后再执行此操作。

### 6.4 安全设置

**密码加盐值**
- 用于密码加密的安全参数
- **修改后所有用户密码将失效**，需要重新设置
- 非必要请勿修改

**登录有效期**
- 设置登录状态保持的天数（默认30天）
- 到期后需要重新登录

## 7. 常见问题FAQ

### Q1: 无法访问系统怎么办？

**检查步骤：**
1. 确认设备已通电，指示灯有亮起
2. 确认您的手机/电脑与设备在同一WiFi网络
3. 确认输入的IP地址正确
4. 如果WiFi不可用，尝试连接设备热点

**如果仍无法访问：**
- 尝试重启设备（断电30秒后重新上电）
- 联系管理员确认IP地址和网络配置

### Q2: 忘记登录密码怎么办？

请联系诗社管理员，管理员可以在系统中为您重置密码。

### Q3: 指示灯一直快速闪烁不停？

这表示设备正在尝试连接WiFi但未成功。可能原因：
- WiFi名称或密码配置错误
- WiFi路由器未开启
- 设备距离路由器太远，信号弱

**解决方法：**
- 连接设备热点进入系统，检查WiFi配置
- 将设备移近路由器
- 重启路由器后再重启设备

### Q4: 系统响应很慢怎么办？

- 检查网络连接是否稳定
- 刷新页面重试
- 如果持续缓慢，可能需要重启设备

### Q5: 发布的诗歌如何修改？

在「藏诗阁」中找到您的作品，点击查看详情后，可以看到「编辑」按钮进行修改。只有作者本人可以编辑自己的作品。

### Q6: 如何获得积分？

积分通过完成「事务」中的任务获得：
1. 在「事务」页面认领任务
2. 完成任务要求的工作
3. 提交完成申请
4. 等待理事审批通过
5. 积分自动发放到您的账户

### Q7: 游客可以做什么？

如果系统开启了游客访问，未登录用户可以：
- 浏览首页、诗歌、活动
- 参与聊天（如已开启）

但无法发布内容或查看敏感信息。

### Q8: 如何联系管理员？

请通过诗社的微信群或其他联系方式与管理员沟通。

## 8. 权限说明

系统设有5个角色等级，权限从高到低：

### 8.1 权限等级表

| 角色 | 说明 |
|-----|------|
| **超级管理员** | 最高权限，可管理所有功能 |
| **管理员** | 可管理大部分功能，但不能修改安全设置 |
| **理事** | 可发起活动、发布任务、审批任务 |
| **财务** | 普通社员权限 + 财务记账功能 |
| **普通社员** | 基础功能：发布诗歌、认领任务、参与聊天 |

### 8.2 功能权限对照表
