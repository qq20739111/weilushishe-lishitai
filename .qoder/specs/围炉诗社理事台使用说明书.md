# 围炉诗社·理事台 使用说明书计划

## 概述
编写一份面向非技术用户的系统使用说明书，内容涵盖系统启动、指示灯状态、功能模块使用、数据备份等内容。

## 说明书结构规划

### 1. 前言
- 系统简介
- 适用对象
- 访问方式（WiFi连接说明）

### 2. 系统启动与指示灯
- 设备启动流程
- LED指示灯状态含义表
  - 快速呼吸（500ms）→ 正在连接WiFi
  - 1秒呼吸 → AP热点模式
  - 2秒呼吸 → WiFi已连接
  - 4秒呼吸 → 双模式运行
  - 快闪一下 → 正在响应操作
  - LED熄灭 → 正常运行（节能模式）

### 3. 连接与登录
- 通过家庭WiFi访问
- 通过热点模式访问（AP模式）
- 用户登录说明
- 忘记密码处理

### 4. 功能模块使用指南
- **首页**：年度热力图、快捷入口、每日推荐
- **藏诗阁**：诗歌浏览、发布新作、编辑/删除
- **活动**：活动查看、发起活动（理事+）
- **摆龙门阵**：实时聊天使用说明
- **事务**：任务认领、完成提交、积分获取
- **财务**：收支明细查看、记账（财务+）
- **社员**：成员名录、个人信息
- **个人中心**：资料修改、密码修改

### 5. 系统管理（管理员）
- 基础设置
- WiFi配置修改
- 自定义字段管理
- 登录日志查看

### 6. 数据备份与恢复（超级管理员）
- 导出备份步骤
- 导入备份步骤
- 注意事项

### 7. 常见问题FAQ
- 无法连接系统
- 忘记密码
- LED异常闪烁
- 系统响应缓慢

### 8. 权限说明
- 各角色权限对照表

## 输出文件
将说明书写入 `doc/用户使用说明书.md`

## 验证方式
- 检查文档格式正确性
- 确保内容准确覆盖所有功能模块
- 语言通俗易懂，适合非技术用户阅读
